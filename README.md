# Java Employee Management System

A simple console-based application developed using Core Java.  

## Features
- Add Employee  
- Update Employee  
- Delete Employee  
- Search Employee  
- View All Employees  

## Technologies Used
- Core Java  
- OOP Concepts  
- Collections Framework (ArrayList)  
- Exception Handling  

## How to Run
1. Clone the repository  
2. Open the project in IntelliJ/Eclipse  
3. Run Main.java
